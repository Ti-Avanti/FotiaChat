# 物品展示功能实现计划

## 功能概述
玩家在聊天中可以展示自己的手持物品、背包内容、末影箱内容，其他玩家可以通过悬浮文本查看或点击打开GUI查看。

## 功能详情

### 1. 手持物品展示 [i]
- 玩家在聊天中输入 `[i]` 占位符
- 替换为物品名称，带有悬浮文本显示物品详情（名称、Lore、附魔等）
- 点击事件可配置（如复制物品名称）

### 2. 背包展示 [inv]
- 玩家在聊天中输入 `[inv]` 占位符
- 替换为可配置的文本（如"[查看背包]"）
- 悬浮文本显示背包概览（物品数量等）
- 点击后打开只读GUI查看该玩家的背包快照

### 3. 末影箱展示 [ec]
- 玩家在聊天中输入 `[ec]` 占位符
- 替换为可配置的文本（如"[查看末影箱]"）
- 悬浮文本显示末影箱概览
- 点击后打开只读GUI查看该玩家的末影箱快照

## 配置文件设计 (config.yml)

```yaml
# 物品展示设置
item-display:
  # 手持物品展示
  hand-item:
    enabled: true
    # 触发占位符
    placeholder: "[i]"
    # 显示格式
    format: "<!i><aqua>[{item_name}]</aqua>"
    # 空手时的显示
    empty-hand: "<!i><gray>[空]</gray>"
    # 悬浮文本格式
    hover:
      - "<!i><white>{item_name}</white>"
      - "<!i><gray>数量: {amount}</gray>"
      - "{lore}"
      - "{enchantments}"
    # 点击事件 (可选: none, suggest_command, run_command, copy_to_clipboard)
    click-action: none
    click-value: ""
    # 权限
    permission: "fotiachat.item.hand"

  # 背包展示
  inventory:
    enabled: true
    placeholder: "[inv]"
    format: "<!i><gold>[查看背包]</gold>"
    hover:
      - "<!i><yellow>点击查看 {player} 的背包</yellow>"
      - "<!i><gray>物品数量: {item_count}</gray>"
    # 点击打开GUI
    permission: "fotiachat.item.inventory"
    # 查看他人背包的权限
    view-permission: "fotiachat.item.inventory.view"

  # 末影箱展示
  enderchest:
    enabled: true
    placeholder: "[ec]"
    format: "<!i><dark_purple>[查看末影箱]</dark_purple>"
    hover:
      - "<!i><light_purple>点击查看 {player} 的末影箱</light_purple>"
      - "<!i><gray>物品数量: {item_count}</gray>"
    permission: "fotiachat.item.enderchest"
    view-permission: "fotiachat.item.enderchest.view"
```

## 技术实现

### 核心类
1. `ItemDisplayManager` - 管理物品展示功能
2. `ItemSnapshot` - 存储玩家背包/末影箱的快照（带过期时间）
3. `ItemDisplayListener` - 监听聊天消息，处理占位符替换

### 快照机制
- 玩家发送消息时，创建背包/末影箱快照
- 快照存储在内存中，带有UUID和过期时间（如5分钟）
- 其他玩家点击时，通过UUID查找快照并打开只读GUI
- 过期的快照自动清理

### 悬浮文本实现
- 使用 MiniMessage 的 `<hover:show_text:'...'>`
- 物品详情包括：名称、数量、Lore、附魔

### 点击事件实现
- 背包/末影箱使用回调系统，点击时执行打开GUI的操作
- 需要在聊天格式中嵌入回调ID

## 权限列表
- `fotiachat.item.hand` - 展示手持物品
- `fotiachat.item.inventory` - 展示背包
- `fotiachat.item.inventory.view` - 查看他人背包
- `fotiachat.item.enderchest` - 展示末影箱
- `fotiachat.item.enderchest.view` - 查看他人末影箱
